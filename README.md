# MedievalEra

## Что это

Это развлечение по проектированию веб версии настольной игры "Эра средневековья". Я не преследую своей целью нарушить авторские права на настольную игру, мне интересен только код.
Сама игра у меня имеется в наличие и я в нее играю с детьми. Блокнот с результатами скоро закончится, нужно где-то хранить их:)

## Этапы готовности

- [ ] Указать имена игроков, количество имеющихся кубиков и их тип. Возможность "бросить кубики" для всех игроков.
- [ ] Реализация переброска кубиков и фиксация значений когда это предусмотрено правилами. Подсчет ресурсов хода.
- [ ] Реализация хода. История бросков.
- [ ] Подсчет результатов настольной игры и хранение истории. Может даже ML.
- [ ] Подсчет ресурсов с учетом стройки (без визуализации игрового поля), история строительства и разрушений.
- [ ] Визуализация поля и взаимодействие с ним
- [ ] Многопользовательская игра
- [ ] Там посмотрим
